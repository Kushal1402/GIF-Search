# GIF-Search
> A GIF search engine, by which user can search any GIF's.

## Built With 
- HTML
- CSS
- JAVASCRIPT
- Giphy API

## Instructions to search 

- Type in the search box anything you want.
- Click on GO button.
- All the related GIF's will be shown.
- User can download & share awesome GIF's.
- Enjoy :smile:
